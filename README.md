## Teaching and Mentorship

At Aptech, I lead instruction across a range of disciplines ‚ÄĒ from foundational web development to advanced full-stack architecture and applied AI. My teaching methodology is project-driven: students do not just study concepts, they implement them under conditions that simulate real professional environments.

**What I focus on in the classroom:**

- Writing clean, maintainable code from the earliest stages of learning
- Understanding architecture before memorizing syntax
- Version control discipline and collaborative workflows
- Bridging textbook theory with deployable, functional applications
- Critical thinking when debugging and designing systems

I invest in students who are serious about their craft. The goal is not to produce graduates ‚ÄĒ it is to produce engineers.

## Selected Projects

<br>

**ATM System**
`C#` `.NET`

A fully functional ATM simulation built in C# with .NET. Implements account authentication, balance enquiry, cash withdrawal logic, transaction logging, and error handling. Designed as a teaching demonstration for object-oriented design patterns and stateful application flow.

<br>

**Laravel CRUD Application**
`PHP` `Laravel` `MySQL`

A structured CRUD platform built on the Laravel framework demonstrating MVC architecture, Eloquent ORM, form validation, route management, and Blade templating. Used in backend development modules to illustrate the full request-response lifecycle in a PHP environment.

<br>

**JSON to HTML Renderer**
`JavaScript` `JSON` `HTML` `CSS`

A client-side utility that dynamically parses structured JSON data and renders it as formatted, styled HTML output. Built to demonstrate DOM manipulation, data traversal, and the relationship between data format and presentation layer ‚ÄĒ a practical exercise in JavaScript fundamentals.
